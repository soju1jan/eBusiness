About BS&Tech

금융 시스템 통합 전문

㈜비에스엔텍은 2015년 설립된 금융 시스템 통합(SI) 전문 기업으로, 은행·증권·카드 등 주요 금융기관의 전산 시스템 신규 구축 및 고도화를 지원해 왔습니다. 10년 이상 금융권 실무 경험을 쌓은 전문가들의 노하우를 바탕으로, 안정성과 효율성을 갖춘 금융 IT 서비스를 제공하고 있습니다.

  • 차세대 전자금융 시스템 구축 (FEP/EAI·채널·계정계 통합)
  • 금융 BPR/PPR/ODS 컨설팅 및 디지털 업무혁신
  • MyData · Open API Aggregator 플랫폼 (데이터 수집·정제·권한관리)
  • 대외연계/정보보호/API Gateway 솔루션 공급
  • ISO 20022 도입·전환(ISO20022 특화 AI모델 개발 공급 : 우리은행 사용중)
  • AI 코파일럿/IDP(문서 이해·자동화) 솔루션 제공

우리의 서비스

전자금융 시스템 구축

은행·증권·카드사의 전자금융 채널 및 차세대 전자금융 시스템을 설계·구축합니다. FEP/EAI, 모바일뱅킹, 인터넷뱅킹 등 다양한 채널과 계정계를 안정적으로 연계합니다.

대외망 연계

금융결제원, 금융감독원, 행안부 등 외부기관과의 안정적 연계 시스템을 제공합니다. API Gateway 기반 아키텍처와 Open API를 함께 구축합니다.

BPR / 페이퍼리스

영업점 스마트창구, 전자약정, 페이퍼리스 업무를 포함한 BPR/PPR 시스템을 컨설팅 및 구축합니다. 고객 경험 개선과 내부 프로세스 효율화를 동시에 달성합니다.

AI Copilot / IDP

파운데이션 AI 모델을 활용해 개발자 지원 코파일럿과 문서지능처리(IDP) 솔루션을 제공합니다. 분석·설계·코드생성·테스트까지 업무 전 과정을 자동화해 생산성과 품질을 향상시킵니다.

Open API · MyData

공공·외부 API, MyData Aggregator 플랫폼을 구축하여 데이터 수집·정제·권한관리·모니터링을 일원화합니다. 금융 데이터 활용 비즈니스 모델을 지원합니다.

검사정보시스템

내부감사·자점검사·상시감시를 지원하는 검사정보시스템을 구축합니다. 데이터 기반의 점검대상 추출, 리스크 모니터링, 컴플라이언스 보고 체계를 제공합니다.
